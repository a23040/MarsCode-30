# 30组成字符串ku的最大次数

### 问题描述

给定一个字符串 s*s*，该字符串中只包含英文大小写字母。你需要计算从字符串中最多能组成多少个字符串 `"ku"`。每次可以随机从字符串中选一个字符，并且选中的字符不能再使用。字符串中的字符大小写可以忽略，即大写和小写字母视为相同。

例如，输入 `"AUBTMKAxfuu"`，从中最多能组成 1 个 `"ku"`。

------

### 测试样例

***样例1：***

> 输入：`s = "AUBTMKAxfuu"`
> 输出：`1`

***样例2：***

> 输入：`s = "KKuuUuUuKKKKkkkkKK"`
> 输出：`6`

***样例3：***

> 输入：`s = "abcdefgh"`
> 输出：`0`
